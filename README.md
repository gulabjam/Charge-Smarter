## Description
Charge Smarter is a minimalist EV companion app that helps drivers quickly find and navigate to the nearest charging stations. Featuring a real-time battery dashboard that monitors your charge level, the app automatically suggests optimal stations based on your remaining range. Its clean map interface renders efficient routes while displaying station locations, availability, and basic details.
